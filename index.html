serust 📡

Serial monitor in Rust

Features ✨

  • Cross platform
  • Read and write by pressing in terminal
  • Auto reconnect
  • Option to Connect product ID! (USB PID)

Installation

To install Serust, please visit the serust website

Usage

You can get info by

serust -h

Build

cargo build --release
